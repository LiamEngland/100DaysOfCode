HTML5 Video

Playing Videos in HTML:

Prior to HTML5, a video could only be played in a browser with a plug-in such as Flash.

The HTML5 video element specifies a standard way to embed a video in a web page.

How video works:

The controls attribute adds video controls, like play, pause, and volume.

Width and height should always be included. If height and width are not set, the page might flicker while the video loads.

The source element allows specification of video files which the browser may choose from. The browser will use the recognised format.

Any text between the tags will only be displayed if browser doesn't support video.

video Autoplay attribute:

The Autoplay attribute starts playing videos automatically.

HTML Video - Methods, Properties and Events:

HTML5 defines DOM methods, properties and events for the video element.

This allows for loading, playing, pausing, duration and volume setting.

There are also DOM events that can notify you weh na video begins to play gets pause, etc.
