CSS RGB Value:

A colour can be specified as an RGB value, using Red, Green and Blue.

Each parameter defines the intensity of the color between values of 0 and 225.

As an example, 0, 115, 0 would be a dark green, as only the green has some value.

To display black, all values must be 0, as for white, all must be 255.

CSS HEX Value:

With CSS, colour can also be specified using a hexidecimal value.

#rrggbb

HEX values work with rr being Red, gg being Green and bb being Blue. These values range between 00 to ff (the same as decimal 0 - 255).

HSL Value:

HSL is another way to choose colours, by using Hue, Saturation and Lightness.

hsl(hue, saturation, lightness)

Hue is a degree on the colour wheen from 0 to 360.0, 360.0 being red, 120 being green and 240 being blue.

Saturation is a percentage value, 0% means a shade of gray and 100% is the full colour.

Lightness is also a percentage, 0% is black, 50% is neither light or dark and 100% is white.

RGBA Value:

RGBA colour values are an extension of RGB colour values with an alpha channel - which specifies the opacity for a colour.

An RGBA colour values is specified as follows:

rgba(red, green, blue, alpha)

The alpha parameter is a number between 0.0 (being fully transparent) and 1.0 (opaque).

RGBA None (Transparent)

HSLA Value:

HSLA colour values are an extension of HSL colour values with an alpha value. The alpha channel specifies the opacity for a colour.

HSLA colours are specified with:

hsla(hue, satruation, lightness, alpha)
