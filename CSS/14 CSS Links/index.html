CSS Links

With the addition of CSS, links can be styled a number of different ways.

Styling Links:

Links can be styled with any CSS property.

I am a Red styled hyperlink.

Links can be styled differently depending on what state they're in.

The four link states are:

  • a:link - a normal, unvisted link.
  • a:visited - a link the user has visited.
  • a:hover - a link when the user's mouse hovers over it.
  • a:active - a link the moment it's clicked.

When setting the styling for several link states, there are some order rules:

  • a:hover - Must come after a:link and a:visited.
  • a:active - Must come after a:hover.

Background Colour:

The background-color property can be used to specify a background colour for links.
