CSS Layout

The display property is the most important CSS property for controlling layout.

The display Property:

The display property specifies if/how an element is displayed.

Every HTML element has a default display value depending on what type of element it is. The default display value for most elements is block or inline.

Block-level Elements:

A block-level element always starts on a new line and takes up the full width available. An example of a block-level element is a div.

I am a block-level element.

Examples of block-level elements are:

  • div
  • h1 - h6
  • p
  • form
  • header
  • footer
  • section

Inline Elements:

An inline element doesn't start on a new line as it only takes up as much width as necessary.

  • span
  • a
  • img

Display: none:

display: none; is often used with javascript to hide and show elements withour needing to delete or recreate them.

Override The Default Display Value:

Every element has a default display value. These values can be overidden.

Changing an inline element to a block element, or vice versa, this can be useful for making pages and elements look a specific way whilst following web standards.

A common example of this usage is the list item element being used for horizontal menus.

  • these
  • are
  • inline
  • list items

Setting a property of an element only changes how the element is displayed, not what kind of element it is. An inline element with display: block cannot have block level elements nested within it.

I am a span element with display set to block.
