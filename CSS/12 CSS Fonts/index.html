CSS Font

The CSS font properties define the font family, boldness, size and style of a text.

CSS Font Families:

In CSS there's two types of font family names:

  • generic family - a group of font families wih a similar look, e.g. Serif or Monospace
  • font family - a specific font family, e.g. Lemon Milk or Montserrat.

Serif fonts have small lines at the ends on some characters.

Sans-serif fonts don't have the small lines at the end of characters.

Monospace fonts have characters that all have the same width.

Font Family:

The font-family property sets the font family of text.

The font-family property should hold serveral font names as a fallback, in case the browser doesn't support the first, or a CDN goes down.

When settings these additional fonts, it should start with the desired and get more generic over time if no others are available.

If a font family has multiple words, they must be declared in quotations marks. When specifying the font family, each font much be separated with commas.

I have a different font family.

Font Style:

The font-style property is mostly used to specify italic text.

The font-style property has three values:

  • normal - The text is shown normally.
  • italic - The text is shown in italics.
  • oblique - The text is leaning in a similar way to italic, however has less browser support.

I have normal font styling.

I have italic font styling.

I have oblique font styling.

Font Size:

The font-size property sets the size of the text.

Adjusting font styles is an essential construct of web design, however should be used correctly. This such as paragraphs should not have styling to make them look like headings, and vice versa.

Propery HTML tags like h1 - h6 should always be used for headings and p for paragraphs.

The font-size value can be absolute or relative.

Absolutely sized text:

  • Sets the text to the specified size.
  • Doesn't allow a user to change the text size in all browsers which can be problematic for accessiblity.
  • Absolute size is useful when the physical size of the output is known.

Relatively sized text:

  • Sets the size relatively to the surrounding elements.
  • Allows a user to change the text size in browsers.

Setting Font Size /w Pixels:

Setting the text size with pixels allows for full control over text size.

I've got my text set to 20 pixels.

Setting Font Size /w Em:

To allow users to resize the text, em is more commonly used rather than pixels. The em size unit is recommended in the W3 standard.

1em is equal to the current font size. The default font size in browsers is 16 pixel, so 1em will equal 16px.

The size can calculated from pixels to em by taking the amount of pixels and dividing that by 16.

My font-size is set in em.

My font is also set in em!

The sizes can of font can be set to their intended or similar sizes, however by setting them in em, it allows for the text to be resized in all browsers.

Thankfully older versions of IE are being phased out, as when resizing, text resizes at an incorrect scale.

Using a Combination of Percent & Em:

By setting the default text size in the body to 100% sets all the text to 100% and can be resized in all browsers.

Font-Weight:

The font-weight property is used to specify the weight of a font. This can be done with preset values or numbers. Examples of preset at 'normal', 'lighter' and for specific a numerical value from 100 - 900 which allows for a lot of control with font weight.

I am using the preset font-weight value of 'lighter.

I am using a specific font-weight value of '900'.

Responsive Font Size:

The text-size can be set with a vw unit, which means "viewport width". By setting text size with this value it'll alter when the viewport size changes.

Viewport is the browser window size, meaning 1vw equals 1% of the viewport width. So each character will take up 1% of the browser window.

Font Variant:

The font-variant property specifies whether or not a text shold be displayed in a small-caps font.

In a small-caps font, all lowercase letters are converted to uppercase letters. However, the converted uppercase letters appear in a smaller font than the original uppercase letters.

I Have Normal Text.

I Have Small Text.

Font Property:

The font property is a shorthand property for the following:

  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family

The font-size and font-family values are required. If one of the other values are missing their default values are used.

The syntax is as follows: font: font-style, font-variant, font-weight, font-size/font-height font-family.
