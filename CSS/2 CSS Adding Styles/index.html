Adding CSS

There's 3 ways to insert a stylesheet:

  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

With an external style sheet, you can change the look of an entire site just by changing a single file.

Internal CSS

An internal style sheet may be used if a single page has a unique style. Internal styling will be defined in the head section.

Inline CSS

Inline styling may be used to apply a unique style for a single element. it can be achieved with the style attribute.
