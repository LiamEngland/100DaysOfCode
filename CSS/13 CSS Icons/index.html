CSS Icons

How to add Icons:

The simplest way to add an icon onto a HTML page, is with an icon library, such as Font Awesome. With these a specific icon class can be added to an element such as i or span.

All listed icons in this entry are scalable and can be customised with CSS.

Font Awesome Icons:

Font Awesome Icons can be accessed from fontawesome, and when signed in, a CDN can be used and added to the head of the page.

Bootstrap Icons

Bootstrap icons are similar and called Glyphicons, these can be added by referencing Bootstrap in the head section of the page. Again as this is a CDN no download is required.

Google Icons:

To use Google icons, the same process can be followed by adding a reference in the head section of the page.
