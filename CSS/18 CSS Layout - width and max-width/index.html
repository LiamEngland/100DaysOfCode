CSS Layout - width & max-width

As mentioned in the previous entry; a block-level element always takes up full width available to it.

Setting the width of a block level element will prevent it from stretching the full width of its container.

With this and setting the margins to auto, the element can be centered while taking up the specified width, and remaining space can split between the two margins.
