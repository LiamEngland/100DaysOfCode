CSS Layout - display: inline-block

The display: inline-block Value:

Compared to display: inline, the major difference is that display: inline-block allows to set a width and height on the element.

Also, with display: inline-block, the top and bottom margins/paddings are respected, but with display: inline they're not.

Compared to display: block, the major difference is that display: inline-block doesn't add a line break after the element. so the element can sit next to other elements.

Below shows different behaviour of display: inline, display: inline-block and display: block.

Using inline-block to Create Navigation Links:

A common use for display: inline-block is to display list items horizontally instead of vertically. Below shows horizontal navigation links:
