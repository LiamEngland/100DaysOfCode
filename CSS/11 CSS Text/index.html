Text Alignment:

The text-align property is used to set the horizontal alignment of text.

Text can be aligned to the left or right, centered or justified.

I am left aligned.

I am center aligned.

I am right aligned.

When text-align is set to 'justify', the line will be stretched so everything is equal width. The margins will have equal width, like in newspapers.

I am a justified text block so my I will fit each line equally. This sentence does have to be very long for it to span the page, however. It's best to have this property set in small containers of text where it'll be of more use.

Text Shadow:

The text-shadow property adds shadow to text. Each property accepts a comma-separated list of shadows to be applied to the text.

The values accepted by the property are:

  • h-shadow - This value is required, it's the position of the horizontal shadow. Negative values are accepted.
  • v-shadow - This value is required, it's the position of the vertical shadow. Negative values are accepted.
  • blur-radius - This value is optional. It's the radius of the blur.
  • color - This value is optional. It sets the colour of the shadow.
  • none - This is the default value, being no shadow.
  • initial - This set the property to its default value.
  • inherit - This inherits the property from its parent.
