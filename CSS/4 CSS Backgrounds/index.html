CSS Backgrounds

The CSS Background properties are used to define the background effects for elements.

Background properties consist of:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

Using text with a background image can sometimes cause problems. If the text is a similar colour or blends with the background it can be harder to read.

CSS background-repeat:

By default, the background-image property repeats an image both horizontally and vertically.

Some images should be repeated only horizontally or vertically, or they will look strange.

An image that fades downward could just repeat on a horizontally, with the background colour being set to the end of time image so becomes a seamless background.

CSS background - shorthand property:

To shorten the amount of code used, it's also possible to specify all the background properties in a single property. This is called a shorthand property.

When using the background shorthand property, the order of property values is:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

If a value is missing, it doesn't matter as long as the others are in order.
