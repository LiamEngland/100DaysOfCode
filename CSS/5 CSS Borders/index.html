CSS Borders

CSS Border Properties:

The CSS border properties allow a border to have the style, width and colour specified.

CSS Border Style:

The border-style property specifies what kind of border to display.

Below are the following values allowed:

  • dotted - Defines a dotted border.
  • dashed - Defines a dashed border.
  • solid - Defines a solid border.
  • double - Defines a double border.
  • groove - Defines a 3D grooved border. The effect depends on the border-color value.
  • ridge - Defines a 3D ridged border. The effect depends on the border-color value.
  • inset - Defines a 3D inset border. The effect depends on the border-color value.
  • outset - Defines a 3D outset border. The effect depends on the border-color value.
  • none - Defines no border.
  • hidden - Defines a hidden border.

CSS Border Width:

The border-width property specifies the width of the four borders.

The width can be set as a specific size (in px, pt, cm, em, etc) or by using one of the three pre-defined values:

  • thin
  • medium
  • thick

The border-width property can have one to four values. These are:

  • top border
  • right border
  • bottom border
  • left border

CSS Border Color:

The border-color property is used to set the colour of the four borders.

The colour can be set by:

  • Name - the preset values such as "green".
  • Hex - using a hex values such as "#000000".
  • RGB - using an RGB value such as rgb(0, 255, 0).
  • Transparent

The border-color property can have from one to four values (for the top border, right border, bottom border and left border).

If border-color is not set, it inherits the color of the element.

Border style works as follows:

Let's say a border-style property has four values: dotted dashed inset outset.

  • Top border is dotted.
  • Right border is dashed.
  • Bottom border is inset.
  • Left border is outset.

If the border-style property has 3 values: dotted solid double.

  • Top border is dotted.
  • Right and Left borders are solid.
  • Bottom border is double.

If the border style propery has two values: dotted solid.

  • Top and Bottom are dotted.
  • Right and Left are solid.

If the border style propery has one value: solid.

  • All Four borders are solid.

The border-style can also work with other properties such as border-width and color.

CSS Border - Shorthand Property:

To shorten the amound of code written, it's possible to specify all the individual border properties in one property.

The border property is a shorthand property for the following individual border properties:

  • border-width
  • border-style (Mandatory)
  • border-color
