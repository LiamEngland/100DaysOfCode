CSS Combinators

CSS Combinators:

In CSS, a combinator is something that explains the relationship between selectors.

A CSS selector can contain more than one simple selector. Between the simple selectors, a combinator canbe included.

There are four different combinators in CSS:

  • descendant selector (Space).
  • child selector (>)
  • adjacent selector (+)
  • general sibling selector (~)

Descendant Selector:

The descendant selector matches all elements that are descenants of a specified element.

div.testClass p - This would select all paragraph elements within div elements that have the .testClass class applied to them.

Child Selector:

The child selector selects all elements that are the children of a specified element.

div > p - This would select all p elements that are children of a div element.

Adjacent Sibling Selector:

The adjacent sibling selector selects all elements that are adjacent siblings of a specified element.

Sibling elements must have the same parent element, and "adjacent" means "immediately following".

div + p - Thisselects all p elements that are places immediately after div elements.

General Sibling Selector:

The general sibling selector selects all elements that are siblings of a specified element.

div ~ p - This example selects all p elements that are siblines of div elements.
