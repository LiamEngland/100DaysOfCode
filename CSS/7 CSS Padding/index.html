CSS Padding

The CSS Padding properties are used to generate space around an element's content, inside any defined borders.

With CSS, you have full control over the padding. There are properties for setting the padding for each side of an element (top, right, bottom, left).

Padding - Individual Sides:

CSS has properties for specifying the padding for each side of an element:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

All the padding propeties can have the following values:

  • length - specifies a padding in px, pt, cm, etc.
  • % - specifies a padding in percentage of the width of the containing element.
  • inherit - as covered in the previous entry, padding will be inherited from the parent element.

Unlike margin, negative values are not allowed.

Padding - Shorthand Property:

Like with Margin, the padding property has a shorthand that can have all 4 properties into one:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Below are how the number of parameters in the shorthand padding property work:

  • Four Parameters: With four, each parameter will set the padding relative to the position. padding: 10px 20px 30px 10px the first will set the top to 10px, second the right to 20px, third the bottom to 30px and last the left to 10px.
  • Three Parameters: With three, the first parameter will set the top padding, the second parameter will set the right and left padding, and the third will set the bottom padding.
  • Two Parameters: With two, the first parameter will set the top and bottom padding, the second parameter will set the right and left padding.
  • One Parameter: With one parameter, all padding will be set to that value.

Padding and Element Width:

The CSS width property specifies the width of the element's content area. The content area is the portion inside the padding, border and margin of the element (the box model).

So, if an element has a specified width, the padding added tothat element will be added to the total width of the element. This is often an undesirable result.


I've a set width and added padding adding to the width

The element has a width of 250px and a padding of 50px, so the total width of the element is 350px due to the padding being applied on both sides.

As this can sometimes be an issue, the box-sizing property can be used. This will maintain the element's width and use the content space for the padding, reducing how much content can be used.
