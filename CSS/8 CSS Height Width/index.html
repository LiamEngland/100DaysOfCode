CSS Height and Width

CSS Setting Height & Width:

The height and width properties are used to set the height and width of an element.

The height and width properties do not includng padding, borders or margins. It sets the height/width of the area inside the padding, border and margin of the element.

CSS Height/Width Values:

The height and width properties may have the following values:

  • auto - This is default. The browser calculates the height and width.
  • length - Defines the height/width in px, cm, etc.
  • % - Defines the height/width in percent of the containing block.
  • initial - Sets the height/width to its default value.
  • inherit - The width/height will be inherited from its parent value.

CSS Setting max-width:

The max-width property is used to set the maximum width of an element.

The max-width property can be specified in length values such as px, cm or in percent of the containing block. By default, the value is set to none, where the where there's no maximum width.

The problem occurs when the browser window is smaller than the width of the element. So a screen 800px wide can't contain a full 1000px width container. When something like this occurs, a scrollbar will appear.

Using max-width instead of having a high width, can allow for smaller screens to to handle a div.
