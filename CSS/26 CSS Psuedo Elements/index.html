CSS Psuedo Elements

Psuedo-Elements:

A CSS psuedo-element is used to style specified parts of an element.

Ways it can be used are:

  • To style the first letter or line of an element.
  • To insert content before, or after, the content of an element.

The ::first-line Psuedo-Element:

The ::first-line psuedo-element is used to add a special style to the first line of text.

The first line psuedo-element can be used to add effects to the first line of text. But because this may be displayed on wide monitors, there needs to be lots and lots and lots and lots and lots and lots of text so that it spans over at least two lines. If this text gets too long for smaller sceens then this may be a lot of text.

One thing to note is this can only be applied to block level elements.

Below are properties that apply to the ::first-line psuedo-element:

  • font properties
  • color properties
  • background properties
  • word-spacing
  • letter-spacing
  • text-decoration
  • vertical-align
  • text-transform
  • line-height
  • clear

One thing to notice is that in psuedo-elements there are two colons, such as ::first-line, whereas psuedo-classes have a single colon, such as :first-line.

The double colon replaced the single-colon notation for psuedo-elements in CSS3. This was an attempt from W3C to distinguish between psuedo classes and elements.

Prior to CSS3, the single colon was used for both classes and elements.

For backward compatibility, the single colon syntax is acceptable for CSS2 and CSS1 psuedo-elements.

The ::first-letter Psuedo-Element:

The ::first-letter psuedo-element is used to add a special style to the first letter of a text.

Below the shows the first letter being formatted to all p elements with the class firstLetter.

The first letter has been styled.

To note, like the ::first-line element, ::first-letter can only be applied to block-level elements.

The following properties that apply to the ::first-letter are:

  • font properties
  • color properties
  • background properties
  • margin properties
  • padding properties
  • border properties
  • text-decoration
  • vertical-align (only if float is set to none)
  • text-transform
  • line-height
  • float
  • clear
