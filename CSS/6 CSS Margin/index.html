CSS Margins

The CSS margin properties are used to create space around elements, outside of any defined borders.

With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom and left).

Margin - Individual Sides:

CSS has properties for specifying the margin for each side of an element:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

All the margin properties can have the following values:

  • auto - The browser calculates the margin.
  • length - Specifiesa margin in px, pt, cm, etc.
  • % - Specifies a margin in % of the width of the containing element.
  • inherit - Specifies that the margin should be inherited from the parent element.

With these values, negative values are allowed.

This has different border values.

Margin - Shorthand Property:

Like others, the margin property has a shorthand property for all the margin properties.

The margin properties being shorthanded are:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

The way the margin property works with four values is like a clock. The values apply clockwise, from top, right, bottom and left.

If the margin has only three values: margin: 20px 40px 20px

With this, the top margin will be 20px, the right and left will be 40px and the bottom will be 20px.

If the margin shorthand has only two values: 20px 30px:

The top and bottom margins will be 20px, the right and left will be 30px.

If the margin shorthand property only has one value, all margins will have that value.

The auto Value:

The auto value can be used to horizontally center an element within its container. The element will take up the width specified, then the margin will split equally between the left and right margins.

I'm a centered div!

The inherit Value:

The inherit value allows an element to inherit the attribute from the parent element.

For example, class div has a margin of 10px, class div2 is nested in class div, and has a margin on inherit. Class div2 will have a margin of 10px as it's inherited the margin from the parent element.

Margin Collapse:

The top and bottom margins on elements can be collapsed into a single margin that's equal to the largest of the two margins.

This can only happen with top and bottom, not right and left:

If element 1 has a bottom margin of 20px and element 2 situated below has a top margin of 10px, they will collapse and only the 20px margin will be used.
