CSS Psuedo Classes

A psuedo-class is used to define a specil state of an element.

It can be used to:

  • Style an element when a user hovers over it.
  • Style visited and unvisited links differently.
  • Style a element when it gets focus.

Psuedo-Class Syntax:

The syntax of a psuedo class is often:

selector:psuedo-class {
  property:value;
}
