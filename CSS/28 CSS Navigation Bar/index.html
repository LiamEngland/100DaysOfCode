CSS Navigation Bar

Navigation Bar - List of Links:

A navigation bar needs standard HTML elements for its base.

A navigation bar is just a styled list of links, which is just using ul and li elements.

Below is an unstyled list.

Below is a styled list.

To achieve the above result, the list-style-type is set to none which removes the bullets.

The margin and padding are set to 0 to remove any default browser settings. This is standard for both vertical and horizontal navigation bars.

Center Links & Add Borders:

Adding text-align: center to li or a to center the links.

Adding the border property to ul elements adds a border around the navbar. If you also want borders inside the navbar, adding border-bottom to all li elements, except the last one.
