HTML Forms



POST should be used with a form when:

  • Data submission is private and does not display data in the address.
  • There is the potential for having large submissions, as the only limit to POST requests are set in the server's configuration.

GET should be used with a form when:

  • Information that isn't sensitive and can be visible in the address.
  • Length of data will be less than 2048 characters.
  • Can be useful for user to bookmark page.
