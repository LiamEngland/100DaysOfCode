The Object Element:

The object element is supported by all browsers.

The object element defines an embedded object within a HTML document.

It's used to embed plug-ins in web pages.

The Embed Element:

The embed element is supported in all major browsers.

The embed element also defines an embeddedobject within a HTML document.

Web browsers have supported the embed element for a long time. However it's no been part of the HTML spec before HTML5.
