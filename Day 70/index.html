HTML 5 Canvas:

The HTML Canvas Element is used to draw graphics, via JavaScript.

The Element is only the canvas to draw the graphics, the drawing must be done via JavaScript.

Canvas Example:

A canvas must have an ID to be accessed via the JavaScript, a width and height to define dimensions. Any styles can be applied using styling.
