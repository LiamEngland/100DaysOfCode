HTML Input Types

Button defines a button.


Checkbox lets the user select Zero or More of a number of options.


Text defines a one-line text input field.


Password defines a password field.


Radio lets the user select one of a number of options


Reset resets all form values to their defaults.


Submit defines a button for submitting form data to a form handler


HTML 5 Input Types

Color is used for input fields that should contain color.


Date is used for input fields containing a date.


Date specifies a date and time input field with no time zone.


Email is used for input fields that should contain an email address.


File defines a file select and browse button for file uploads.


Month allows user to select a month and year.


Number defines numeric input field.


Range defines a control for entering a number whose exact value is not important.


Search is used for search fields. A search field behaves like a regular text field.


Tel is used for input fields that should contain a telephone number.


Time allows the user to select a time (no time zone).


Url is used for input fields that should contain a URL address.


Week allows the user to select a week and year.
